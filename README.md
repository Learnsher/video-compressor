# 🔒 安全影片壓縮工具

100% 客戶端處理，影片不會上傳到任何伺服器。

## 使用方法

1. 打開 `index.html`（本地或部署到 GitHub Pages）
2. 上傳影片
3. 選擇壓縮選項
4. 下載壓縮後的影片

## 部署到 GitHub Pages

1. 建立 GitHub repository
2. 上傳這3個檔案
3. Settings → Pages → Source: main branch
4. 訪問 `https://你的用戶名.github.io/repo名稱`

## 安全性

- ✅ 所有處理在瀏覽器完成
- ✅ 無後端伺服器
- ✅ 無資料收集
- ✅ 開源代碼可審查
